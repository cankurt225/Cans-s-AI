# Cans-s-AI
Bu repo Atatürk Üniversitesi AYZEK kulübü tarafından düzenenen hackathon projesi için oluşturulmuştur.

App klasörü içerisinde Flutter ile geliştirdiğimiz arayüzümüz bulunmaktadır. (Zip halinde Wp üzerinden paylaşılmıştır)

Server klasörü içerisinde django backend server bulunmaktadır. ( Zip halinde Wp üzerinden paylaşılmıştır)

App klasörü içerisindeki Flutter App'i çalıştırmak için:
  Flutter.dev’den sdk edinimi gereklidir.
  Kurulum kontrolü için “flutter doctor” komutunu çalıştırınız.

Server klasörü içinde python django sunucusunu çalıştırmak için:
  Python kütüphaneleri kurulmalıdır. 
  “pip install -r requirements.txt” kodu ile kurunuz
  Modeller django sunucusunda çalışmaktadır. main.py dizinine girerek aşağıdaki kodu çalıştırınız.
  “python manage.py runserver”
